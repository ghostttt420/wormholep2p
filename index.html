/// HOW DOES THIS WORK?
NetLink 95 uses WebRTC technology to create a direct data pipe between two devices. Unlike cloud storage (Google Drive, Dropbox), your file is never uploaded to a server. It streams directly from your device to the recipient's device.
/// IS IT PRIVATE?
Yes. The connection is End-to-End Encrypted. Since there is no middleman server storing your file, nobody—including us—can see what you are sending.
/// DO I NEED INTERNET?
You need a brief internet connection to "handshake" (find the other device). Once connected, if both devices are on the same WiFi, data transfers locally. If on different networks, it uses the internet.
